# campus-energy-dashboard-Arth-Rana

This project analyzes electricity consumption from multiple campus buildings and generates a complete end-to-end data processing pipeline.
It reads raw CSV files, validates them, performs aggregations, applies object-oriented modeling, generates visual dashboards, and exports summary reports for administrative decision-making.

Features
	•	Automatic CSV ingestion from /data/
	•	Daily & weekly aggregation (Pandas)
	•	OOP modeling: Building, MeterReading, BuildingManager
	•	Matplotlib dashboard: line, bar, scatter charts
	•	Export cleaned CSVs and summary report
